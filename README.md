# TDD practice

This a educational project made to practice test driven development

## Built With

- Ruby

## Getting Started

- To get a local copy up and running follow these simple example steps.

### Prerequisites

- Ruby

### Setup

- Clone the GitHub Repository
- Go to the Project Directory
- run `bundle install`

### Usage

- Run `rspec`
